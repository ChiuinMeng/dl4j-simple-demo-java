# dl4j-simple-demo-java
智能系统需要做作业，便去找了dl4j来学习。项目里主要是一些demo，来源于dl4j官网给的案例，只做了些注释工作，并尝试修改了一些代码。

因为官网给examples过于庞大，build都要好久，还不一定能成功。对于新入门深度学习的新手来讲很不友好！

本项目的5个可运行案例都弄懂的话，还感兴趣，再去下官网的案例进一步学习吧。加油！
